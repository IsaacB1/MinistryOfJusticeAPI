# Task Tracker Application

A fullstack web application for managing tasks.

---

### Built With

**Backend:**
- Spring Boot & Java
- H2 in-memory database

**Frontend:**
- HTML & CSS
- React (with Vite)
- TypeScript

---

## Prerequisites

Make sure you have the following installed:

- Java 17+
- Maven (for Spring Boot)
- Node.js and npm (for frontend)
